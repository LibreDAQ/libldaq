# LibreDAQ official repository

LibreDAQ is a versatile, modular and hackable Data Acquisition (DAQ) platform. 
This repository includes the sources for: 
* Circuit schematics
* C++ SDK
* C++ examples
* Firmware
* ROS node
* LibreDAQStudio
